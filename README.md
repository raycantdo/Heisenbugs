# Study Tracker 
A C++-based study assistant that helps students manage their courses,tracks study sessions and organize academic tasks.It records daily, weekly, and term-wise study hours, logs class topics,assignments and exams, and provides a to-do list for planning study tasks. The system also highlights under-studied courses and shows useful insights like most focused time, average session length.
